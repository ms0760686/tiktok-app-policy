隱私權政策 Privacy Policy

最後更新 Last updated: 2026-05-21

1. 概述 Overview

本隱私權政策說明本應用程式（TikTok Content Posting API 應用）如何收集、使用及保護您的個人資訊。我們重視您的隱私，致力於透明化資料處理方式。

This Privacy Policy describes how this application (TikTok Content Posting API application) collects, uses, and protects your personal information. We are committed to transparency in data handling.

2. 收集的資料 Data We Collect

資料類型說明用途
TikTok Access Token授權本應用程式代表您操作 TikTok API影片上傳授權
影片檔案您提供的 AI 生成影片（本地或 URL）上傳至 TikTok
TikTok Publish ID上傳後 TikTok 回傳的識別碼追蹤上傳狀態

我們不收集： 密碼、個人識別資訊、瀏覽記錄、位置資料、或任何超出 API 操作必要範圍的資料。

We do NOT collect: passwords, personal identification data, browsing history, location data, or any data beyond what is necessary for API operations.

3. 資料使用方式 How We Use Data

收集的資料僅用於：

  • 透過 TikTok Content Posting API 上傳影片至您的 TikTok 帳號
  • 追蹤上傳狀態（publish_id）
  • Token 自動更新（如有提供 Refresh Token）

我們不會將您的資料出售、出租或分享給任何第三方。

We do not sell, rent, or share your data with any third parties. Data is used solely for the video upload functionality.

4. TikTok API 資料處理 TikTok API Data

本應用程式使用 TikTok Content Posting API。透過 API 傳輸的資料同時受 TikTok 隱私權政策約束：

TikTok Privacy Policy

本應用程式存取的 TikTok API 權限：

  • video.publish — 代表您發布影片
  • video.upload — 上傳影片至草稿匣

5. 資料儲存與安全 Data Storage & Security

Access Token 及 Refresh Token 儲存於應用程式的本地設定檔（.env 檔案）中，不會上傳至任何外部伺服器。您有責任保護這些認證資訊的安全。

Access Tokens are stored locally in the application configuration file. They are never transmitted to external servers beyond the official TikTok API endpoints.

6. 資料保留 Data Retention

應用程式不持久化儲存影片內容。影片上傳後，本地副本由您自行管理。TikTok 平台上的影片受 TikTok 的資料保留政策管轄。

The application does not persistently store video content. Videos on the TikTok platform are governed by TikTok's data retention policies.

7. 使用者權利 Your Rights

您可以隨時撤銷本應用程式的 TikTok 授權，方法：TikTok 應用程式 → 設定 → 安全 → 已授權應用程式。

You may revoke this application's TikTok authorization at any time: TikTok app → Settings → Security → Authorized Apps.

本政策如有變更，將更新本頁面的「最後更新」日期。建議定期查看本頁面。

If this policy changes, we will update the "Last updated" date on this page.
