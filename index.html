AI Video Publisher

Automated AI-generated video uploads to TikTok via Content Posting API

應用程式說明 About This App

AI Video Publisher 是一套 AI 影片自動化發布工具。它能將 AI 生成的短影音（透過 Kling 等 AI 影片生成服務產生）自動上傳至 TikTok 帳號。 上傳後影片先存入創作者草稿匣（Inbox），由帳號擁有者審核後手動發布，確保內容品質。

AI Video Publisher is an automation tool for publishing AI-generated short videos to TikTok. It generates videos using AI services and uploads them to the creator's TikTok Inbox (draft mode) for manual review before publishing.

工作流程 How It Works

  1. 使用 AI 影片生成服務（如 Kling AI）製作短影音內容
  2. 透過 TikTok Content Posting API 將影片上傳至創作者草稿匣
  3. 帳號擁有者在 TikTok 應用程式中審核草稿，確認後手動發布
  4. 追蹤 Publish ID 並記錄上傳結果

TikTok API 權限 Requested Permissions

  • video.upload — 上傳影片至草稿匣 Upload video to creator inbox
  • video.publish — 從草稿直接發布（選用）Direct post from draft (optional)

本應用程式僅代表帳號擁有者本人操作，不存取其他使用者資料。
This app operates only on behalf of the account owner and does not access other users' data.

資料安全 Data & Privacy

  • Access Token 僅儲存於本地設定檔，不上傳至第三方伺服器
  • 影片內容不被本應用程式持久化儲存
  • 不收集任何使用者個人識別資訊
